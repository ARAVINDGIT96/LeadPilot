# Lead Pilot

Lead Pilot is an AI-powered business lead generation and outreach platform. It helps users find business leads, get AI-based suggestions, and send emails directly from the application.

## Features

* User Authentication
* Business Lead Search
* AI Suggestions
* Email Automation
* Lead Management
* Real-Time Notifications

## Technologies Used

* React.js
* FastAPI
* PostgreSQL (Supabase)
* Auth0
* Google Places API
* Gmail API
* Gemini AI
* Tailwind CSS

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Login to the application.
2. Search for business leads.
3. View AI-generated suggestions.
4. Send emails to selected leads.
5. Save and manage leads.

## Project Goal

To simplify business lead generation and outreach by combining AI capabilities with external API integrations.
